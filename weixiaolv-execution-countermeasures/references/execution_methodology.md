## 二、四个指导方针（扩大主体路径总纲）

### 方针 1：扩大主体
- **定义**：通过追加/变更被执行人，将执行义务主体从债务人本人扩展至关联主体
- **两条路径**：
  - 执行程序内直接追加（28 种）→ 见 `28_direct_expansion.md`
  - 另案诉讼穿透追加（52 种）→ 见 `52_litigation_expansion.md`
- **核心思维**："拿起锤子找钉子"——先掌握武器（追加路径），再锁定目标（债务人软肋）

### 方针 2：扩大责任财产
- **定义**：穷尽一切手段挖掘债务人名下及隐匿的财产线索
- **三维挖掘**：
  - 显性财产（银行账户、房产、车辆、股权）
  - 隐性财产（代持资产、应收债权、到期收益）
  - 跨境财产（境外账户、信托资产）

### 方针 3：排除执行障碍
- **定义**：清除案外人异议、租赁权抗辩等阻挡执行的因素
- **主要障碍类型**：
  - 虚假租赁（买卖不破租赁滥用）
  - 虚假买卖（案外人提异议）
  - 共有财产分割

### 方针 4：追究拒执罪刑事责任
- **定义**：以拒不执行判决、裁定罪刑事追责形成高压态势
- **核心路径**：刑法第 313 条；法释〔2024〕13 号；法发〔2025〕8 号

## 三、思维方式（执行律师认知框架）

### 1. 发散思维
> "切记勿用以我们的认知水平认知被执行人"
- 不以常人逻辑推断债务人行为
- 债务人的规避手段往往超出律师的日常经验
- **操作**：遇到瓶颈时，主动寻找"最不可能但最有效"的路径

### 2. 系统思维（头脑风暴）
- 将案件要素全部列出，进行关联性分析
- 从债务人社会关系网、资金流向链、资产转移路径三个维度展开

### 3. 假设思维
> "大胆设想、积极求证、刨根问题、不达目的不罢休"
- 先提出假设（债务人有某类资产/转移了某类财产）
- 再设计验证路径（查询哪些信息可以证明或推翻假设）
- 反复迭代，直到锁定确定性线索

### 4. 串联思维
- 将分散的线索串联成完整的证据链
- 典型串联：手机号 → 微信/支付宝实名 → 关联企业 → 股权穿透 → 实际控制人

### 5. 举一反三，法无定法
- 每发现一个线索，立即思考：该线索可能指向哪 3 类资产/哪 5 个关联主体？
- 不拘泥于固定套路，根据实际情况灵活应变

## 四、律师三查体系（实操作业标准）

### 第一查：网查（线上情报）

#### 自然人
| 查什么 | 怎么查 | 证据价值 |
|--------|--------|---------|
| 涉诉裁判文书 | 中国裁判文书网、北大法宝 | 挖掘未披露的债务/关联人 |
| 基本信息（姓名/电话/地址/身份证）| 百度搜索、百度地图 | 定位财产线索 |
| 手机号（核心抓手）| 微信添加、支付宝转账验证 | 确认实名 → 锁定账号/关联企业 |
| 抖音/快手/微博 | 搜索姓名简写、网名 | 视频暴露位置/资产线索 |
| 保险 | 银保监会官网查询、各大险企小程序 | 人寿保险保单现金价值 |
| 婚姻信息 | 诉讼卷宗、执行卷宗（送达地址）| 挖掘配偶财产 |
| 户籍信息 | 律师调查令 | 确认家庭成员、房产线索 |

#### 法人
| 查什么 | 怎么查 | 证据价值 |
|--------|--------|---------|
| 工商信息 | 企查查/天眼查/国家企业信用信息公示系统 | 股东、出资、变更、经营异常 |
| 涉诉裁判文书 | 裁判文书网 | 挖掘关联交易、隐性债务 |
| 知识产权 | 中国商标网、专利审查信息查询 | 高价值商标可执行 |
| 政府采购/中标 | 中国政府采购网 | 现金流线索 |
| 税务信息 | 律师调查令→税务局 | 实际经营规模 |
| 社保缴纳 | 律师调查令→社保局 | 员工人数→经营规模 |

### 第二查：线下调查（现场 + 律师调查令）

| 场景 | 目标 | 律师调查令可查 |
|------|------|--------------|
| 不动产 | 实际占有人、产权状态 | 房产底档、抵押登记 |
| 车辆 | 实际占有使用人 | 车辆登记信息 |
| 银行账户 | 资金流水 | 银行流水（执行阶段须法院调取）|
| 证券账户 | 持仓情况 | 证券公司客户信息 |
| 保险 | 保单现金价值 | 保险公司客户信息 |
| 工商档案 | 股东出资、董监高 | 完整工商内档（含验资报告）|
| 婚姻档案 | 夫妻共同财产 | 婚姻登记信息 |

### 第三查：情报链穿透

典型穿透路径：
```
手机号 → 微信/支付宝实名 → 关联企业A → 股权穿透A的股东 → 自然人B →
B的另一个手机号 → 微信/支付宝 → 关联企业C → 循环
```

## 五、找人找钱九大基本动作

| 编号 | 动作 | 描述 | 适用场景 |
|------|------|------|---------|
| 1 | 查银行流水 | 调取目标银行账户资金流水 | 查隐匿资产、抽逃出资、资金混同 |
| 2 | 查不动产底档 | 律师调查令查房产登记信息 | 查实际产权人、抵押情况、查封状态 |
| 3 | 查股权代持 | 穿透股权背后的真实持有人 | 查代持、查金蝉脱壳 |
| 4 | 查婚姻档案 | 挖掘夫妻共同财产线索 | 假离婚逃债、追加配偶 |
| 5 | 查户籍信息 | 确认家庭成员、居住地址 | 查未成年子女名下财产、查亲属关系 |
| 6 | 查社保缴纳 | 确认劳动关系、工资发放主体 | 查实际用工主体、查工资账户 |
| 7 | 查手机号关联 | 通过手机号关联各平台账号 | 查微信/支付宝余额、查社交账号 |
| 8 | 查税务数据 | 确认真实经营规模、收入 | 查隐匿收入、查逃税线索 |
| 9 | 查知识产权 | 商标/专利可作为执行标的 | 查高价值无形资产 |
