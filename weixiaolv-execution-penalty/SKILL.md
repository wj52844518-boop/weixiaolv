# 技能：拒不执行判决、裁定罪刑事追责与自诉启动

> 版本：V1.0 | 依据：法释〔2024〕13号 + 法发〔2025〕8号 | 施行：2024-12-01 + 2025-07-01
> 架构：原子化 Skill（trigger_checker + route_selector + evidence_mapper 三引擎联动）

---

## 一、核心职责

作为韦小律拒执罪追刑原子 Skill，本模块负责：

1. **实体审查**：检测被执行人逃债行为是否满足"情节严重"标准（2024解释第三条）
2. **程序路由**：根据当前程序状态，精准匹配合法追刑路径（2025意见第15-18条）
3. **证据映射**：按刑事"排除合理怀疑"标准，评估证据链完整度并补全缺口
4. **文书交付**：全自动装配刑事自诉状/控告书/移送申请书三套文书

---

## 二、三引擎联动流程

```
[输入：case.json 执行状态快照]
    │
    ▼
【引擎一：trigger_checker.py】
  目的：检测"情节严重"特征码 + 计算犯罪嫌疑评分
  输入：case.json execution_state.facts + violations_detected
  输出：matched_triggers[] + overall_score + prosecution_ready
    │
    ▼ prosecution_ready == true 时
【引擎二：route_selector.py】
  目的：确定合法追刑路径（公诉 vs 自诉）
  输入：substantive_completeness + police_status + is_court_supportive
  输出：route ∈ {NONE / COURT_PUBLIC_TRANSFER / PUBLIC_REPORT_FIRST / PRIVATE_PROSECUTION}
    │
    ▼ route 确定后
【引擎三：evidence_mapper.py】
  目的：按刑事标准评估证据链 + 生成调查令补全缺口
  输入：known_evidence + matched_triggers
  输出：chain_health + gaps[] + investigation_applications[]
    │
    ▼
【文书装配：根据 route 匹配 templates/】
  route == PRIVATE_PROSECUTION  → 拒不执行判决裁定罪刑事自诉状.docx
  route == PUBLIC_REPORT_FIRST  → 刑事控告书_公安机关报案用.docx
  route == COURT_PUBLIC_TRANSFER → 移送公安机关立案侦查申请书.docx
```

## 三、四大构成要件（实体法基础）

| 要件 | 内容 | 核心证据 |
|------|------|---------|
| **一、负有执行义务** | 被执行人对申请执行人有执行义务（生效法律文书确认）| 判决书副本 + 执行通知书 + 送达回证 |
| **二、有能力执行** | 被执行人有能力履行（银行账户/房产/车辆/高消费记录）| 财产线索 + 高消费记录 |
| **三、拒不执行** | 被执行人采取积极或消极方式拒不执行 | 转移行为/虚假报告/违反限高 |
| **四、情节严重** | 上述行为达到"情节严重"门槛（2024解释第三条）| 金额/手段/后果综合认定 |

---

## 四、2024解释十大"情节严重"特征码

| 代码 | 情形 | 法律依据 | 证据要点 |
|------|------|---------|---------|
| T1 | 隐藏/转移/低价转让财产 | §3(一) | 过户协议+银行流水 |
| **T1.5** ⭐ | **判决生效前即开始转移财产** | §3(一)新增 | 过户时间节点 vs 判决生效时间对比 |
| T2 | 虚假诉讼/仲裁/调解 | §3(二) | 串通证据+诉讼材料时间线 |
| T3 | 拒不交出财产/证照 | §3(三) | 搜查记录+拒交证据 |
| T4 | 司法拘留后仍不执行 | §3(四) | 拘留决定书+拘留后仍不履行 |
| T5 | 违反限高经罚款后仍高消费 | §3(五) | 限高令送达回证+高消费记录 |
| T6 | 虚假报告/伪造证据 | §3(六) | 虚假申报表+伪造材料 |
| T7 | 转移共有财产/赠与 | §3(七) | 过户记录+亲属关系证明 |
| T8 | 其他情节严重 | §3(八) | 综合证据 |
| TS1-3 | 情节特别严重（3-7年）| §4 | 数额巨大（个人≥100万）+暴力抗拒 |

**T1.5是2024解释最大亮点**：犯罪时间前移至判决生效前，打通了"诉前转移"的追刑路径。

## 五、追刑路径决策（2025意见第15-18条）

### 三条路径的选择逻辑

```
实体构成要件完成度 ≥ 70% ？
  ↓ 否 → 阻断：先补齐证据（触发 investigation-hound）
  ↓ 是
  已向公安报案？
    ↓ 否
    执行法院配合度高？
      ↓ 是 → 路径B：法院移送公安（COURT_PUBLIC_TRANSFER）
      ↓ 否 → 路径C：向公安报案（PUBLIC_REPORT_FIRST）
    ↓ 是（已报案）
    公安超期30天未答复 OR 已出具不予立案 OR 检察院不起诉？
      ↓ 是 → 路径A：刑事自诉解锁（PRIVATE_PROSECUTION）
      ↓ 否 → 等待30天窗口期（WAIT_30DAY_WINDOW）
```

### 路径详情

| 路径 | 适用情形 | 法定前置条件 | 目标文书 |
|------|---------|------------|---------|
| **A：刑事自诉** | 公安30天不答复/不予立案/检察院不起诉 | 三选一 | 刑事自诉状 |
| **B：法院移送公安** | 执行法院配合度高+未向公安报案 | 无 | 移送申请书 |
| **C：公安控告（起步）| 任何情形（起步路径）| 无 | 刑事控告书 |

## 六、证据链完整度评估

**刑事"排除合理怀疑"标准**（高于民事优势证据标准）：

| 要件 | 证据要素 | 证明标准 | 举证方 |
|------|---------|---------|---------|--------|
| 负有执行义务 | 判决书+执行通知书+送达回证 | 排除合理怀疑 | 自诉人 |
| 有能力执行 | 财产线索+高消费记录 | 排除合理怀疑 | 自诉人 |
| 拒不执行 | 转移行为+虚假报告+违反限高 | 排除合理怀疑 | 自诉人 |
| 情节严重 | 金额+手段+后果 | 排除合理怀疑 | 自诉人 |

**证据链健康度**：
- ≥ 70%：可启动追刑（自诉/公诉）
- 50-69%：先申请律师调查令补齐证据
- < 50%：触发 investigation-hound 继续搜集线索

## 八、关键坑（Gotchas）

1. **30天起算日**：EMS寄出日（保留寄件人存联）= 起算日，非签收日
2. **自诉前置三选一**：公安不予立案 OR 30天超期 OR 检察院不起诉，缺一不可
3. **T1.5黄金窗口**：2024解释新增——判决生效前转移也可构成拒执罪
4. **主体扩展**：实际控制人/控股股东可被追诉（2024解释第二条）
5. **同步申请财产保全**：自诉立案同时申请对被告人采取强制措施
