# 2025年两高一部意见：法发〔2025〕8号
## 《关于办理拒不执行判决、裁定刑事案件若干问题的意见》
### 施行日期：2025年7月1日（最高人民法院、最高人民检察院、公安部联合发布）

---

## 一、核心架构：公诉为主、自诉为辅

2025意见明确了中国拒执罪追刑的基本格局：
- **公诉优先**：公安机关侦查 → 检察院审查起诉 → 法院判决
- **自诉补充**：满足特定法定条件时，申请执行人可直接向法院自诉

---

## 二、第一章：管辖与主体（第1-8条）

| 条款 | 内容 | 实战要点 |
|------|------|---------|
| 第1条 | 拒执罪案件由**执行法院所在地公安机关**管辖 | 管辖地=执行法院所在地，非被告人住所地 |
| 第2条 | 多人共同犯罪的，各自管辖或协商管辖 | 若涉及跨省转移财产，可协调管辖 |
| 第3条 | 执行法院发现涉嫌拒执罪，应**移送**公安机关 | "应当"=义务性规范，法院不得拒绝 |
| 第4条 | 执行机构收集犯罪线索后移交侦查 | 执行法官有主动收集义务 |
| 第5-8条 | 证据标准与公安机关调查取证权限 | 公安可使用技术侦查/隐匿身份侦查 |

---

## 三、第二章：公诉程序（第9-14条）

### 报案与受理流程
```
律师向执行法院所在地公安机关经侦部门报案
    ↓（提交《刑事控告书》+证据材料）
公安机关出具《受案回执》（必须要求！）
    ↓
公安审查（30日内决定是否立案）
    ↓
┌─ 立案 → 侦查 → 移送检察院 → 起诉 → 判决
└─ 不予立案 → 出具《不予立案通知书》→ 进入自诉路径
```

### 公安机关30天审查规则
- **原则**：接受控告后30日内决定是否立案
- **超期处理**：30日内未书面答复 → 控告人可凭EMS存联申请自诉
- **实务技巧**：律师须在第28天开始跟进，保留催办记录

---

## 四、第三章：自诉程序（第15-18条）⭐ 核心

### 第15条：自诉法定前置条件（三选一）

| 条件 | 触发自诉需要的证据 | 实战常用程度 |
|------|------------------|------------|
| ①公安不予立案并出具《不予立案通知书》 | 不予立案通知书原件 | ⭐⭐⭐ 常用 |
| ②检察院对被执行人作出不起诉决定 | 不起诉决定书 | ⭐⭐ 较少 |
| ③公安接受控告材料后**30日内未书面答复** | EMS寄件存联 + 受案回执 | ⭐⭐⭐⭐⭐ **最常用** |

### 第16条：自诉状的形式要求
- 须列明被告人基本信息
- 须附具上述三项前置条件的证明材料
- 须说明具体的拒执行为（时间/地点/行为/后果）

### 第17条：法院对自诉案件的审查
- 法院应在15日内决定是否立案
- 立案庭审查"形式要件"（前置条件+自诉状格式）
- 不审查"实体是否构成犯罪"（那是庭审的任务）

### 第18条：自诉审理的特殊规则
- 自诉人可以申请财产保全
- 法院可对被告人采取强制措施
- 自诉审理期间，公诉机关可以另行追诉（一事不再理原则的例外）

---

## 五、第四章：证据规格（第19-25条）

| 证据类型 | 刑事标准 | 实务要求 |
|---------|---------|---------|
| 书证 | 原件>复印件 | 提供原件或加盖公章的复印件 |
| 电子数据 | 须符合电子取证规范 | 微信记录须全程录制+录屏+时间戳 |
| 证人证言 | 须有身份证明 | 证人须出庭作证或提供书面证言+公证 |
| 鉴定意见 | 须由有资质机构出具 | 财产评估须由法院入库评估机构出具 |

## 六、程序流程图（2025意见完整版）

```
[执行立案]
    │
[执行受阻/发现逃债行为]
    │
[律师准备证据+起草《刑事控告书》]
    │
┌─ 路径A：执行法院移送（法院主动发现或律师申请）
│   → 执行法院制作《移送函》
│   → 移送公安机关立案侦查
│   → 公安立案侦查 → 检察院审查起诉 → 法院判决
│
├─ 路径B：律师向公安控告（起步路径）
│   → 公安出具《受案回执》
│   → 30天审查期满
│   → ┌─ 公安立案 → 走公诉程序
│   → └─ 公安超期不答复 → 律师持EMS存联申请自诉
│   → ┌─ 公安不予立案 → 律师持《不予立案通知书》申请自诉
│   → └─ 检察院不起诉 → 律师持《不起诉决定书》申请自诉
│
└─ 路径C：直接向执行法院自诉（满足第15条前置条件时）
    → 提交自诉状+前置条件证明材料
    → 执行法院立案庭15日内决定立案
    → 立案后开庭审理
```

| 施行日期 | 2024-12-01 | 2025-07-01 |
| 发布主体 | 两高 | 两高一部 |
| 侧重点 | 犯罪构成+情节标准 | 管辖+公诉+自诉+证据 |
